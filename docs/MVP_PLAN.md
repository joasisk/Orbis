# MVP Plan

## MVP objective
Deliver a private, self-hosted, usable system that supports:
- project/task tracking
- AI weekly planning
- reminders
- focus mode
- wife visibility

## Included capabilities

### Core data and workflow
- Areas of Life, Projects, Tasks
- recurring commitments
- deadlines, priority, effort, energy
- private vs visible tasks
- wife influence values

### Weekly plan
- proposal generated on Sunday evening
- approval workflow before applying plan

### Daily use
- focus mode
- start/stop/sidetrack/unable-to-finish flows
- simple reminder engine
- overload and reset prompts

### Integrations
- one calendar integration path
- one note ingestion path
- one AI provider abstraction layer with support for multiple providers

## MVP cuts
- GraphQL
- native mobile apps
- advanced analytics
- full two-way sync with all tools
- plugin marketplace
- complex social/collaboration models

## MVP success criteria
- user can capture and organize active work
- user receives a usable weekly plan
- user can start and track tasks daily
- spouse can see relevant schedule context and influence household-critical prioritization
- system remains low-friction and non-overwhelming
